- Match:        var(--cl-<token>, #<hex>)
- Match regex:  var\(--cl-[a-z-]+,\s*#[0-9A-Fa-f]+\)
- Replace with: var(--cl-<token>)
- Critical:     MULTIPLE matches per line are common — Pitfall 1. Use a global

- Out of scope: rgba(...) fallbacks (different prefix — gate regex requires `#`)
                stay untouched. Bare hex literals NOT preceded by `var(--cl-` stay
                untouched. Color literals like `color: #fffdf8;` (not inside a
                var() fallback) stay untouched.

- Match:        var(--cl-error, #B23B2C)
- Replace with: var(--cl-danger)
- Why rename:   --cl-error is NOT defined in the canonical block (Plan 01).
                The closest canonical equivalent is --cl-danger (value #B54C36
                ≈ #B23B2C, both fault-clay red). Without the fallback and without
                the rename, the color won't resolve.
