## Visuals

- **Focal Point**: The conversation trace and the primary action buttons (e.g., 'Send Draft', 'Ask Question') act as the primary visual anchors.
- **Accessibility**: All icon-only actions (if any) must include label fallbacks (`aria-label` or `.sr-only` text) for screen readers.

---

## Color

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | #F5F0E6 (Trailpaper) / #101614 (Dark) | App/page background, main canvas |
| Secondary (30%) | #FBF7EE (Warm Stone) / #18211F (Dark) | Cards, panels, sidebar, nav |
| Accent (10%) | #A94F30 (Path Copper) / #D98A4A (Dark) | Primary actions, active route marks |
| Destructive | #B54C36 (Fault Clay) / #E18C7D (Dark) | Blocked, failed, risky, policy denied |

Accent reserved for: Primary action buttons, active route markers, selected items, focus rings. Never for full-page backgrounds or generic secondary actions.
