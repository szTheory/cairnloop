<task type="checkpoint:human-action" gate="blocking">
  <what-needed>Manual Hex.pm publish for v0.1.0 and Git Tagging</what-needed>
  <why>Hex v2.4 mandates interactive 2FA for key generation/first publish. The agent cannot automate the interactive browser flow (D-02).</why>
  <how-to-do-it>
    1. Run `mix hex.info cairnloop` to verify the package name is unclaimed. Stop immediately if it is claimed.
    2. Tag the release: `git tag v0.1.0`
    3. Push the tag: `git push origin v0.1.0`
    4. Authenticate to Hex if needed: `mix hex.user auth`
    5. Publish the package: `mix hex.publish` (follow the interactive prompts).
  </how-to-do-it>
  <resume-signal>Type "done" when the package is published to hex.pm</resume-signal>
</task>
