## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| browser -> local filesystem | Playwright opens `brandbook/index.html` by `file://` and monitors all asset requests. |
| user input -> theme state | User interaction toggles only local `data-theme` and optional non-sensitive localStorage. |
| viewport -> layout | Browser rendering can reveal clipped, blank, overlapping, or unreachable content that source checks miss. |

| T-51-11 | Denial of Service | file-url rendered page | mitigate | Assert nonblank body, required visible sections, viewport bounding boxes, and no major document overflow across mobile/tablet/desktop. |
