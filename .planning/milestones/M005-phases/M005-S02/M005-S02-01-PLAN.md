  3. Define the file contents as a string containing a standard `Telemetry.Metrics` configuration block.

     - `cairnloop.support_csat_score` (summary) matching `[:cairnloop, :feedback, :csat, :stop]` (metadata: `:rating`)
  5. CRITICAL: For resolution time and CSAT, you MUST use an anonymous function for the `measurement` option to extract values from `metadata` (e.g. `fn _measurements, metadata -> Map.get(metadata, :business_duration_seconds, 0) end`), as `telemetry_metrics` only checks the `measurements` map by default if an atom is provided.
  6. Use `Igniter.Project.Module.create_module/4` to inject the module into the host app.
