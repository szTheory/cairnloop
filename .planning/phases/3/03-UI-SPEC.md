## Color

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | `var(--cl-bg)` | Application background, subtle modal backdrop overlay |
| Secondary (30%) | `var(--cl-surface)` | Modal surface background, result item surface |
| Accent (10%) | `var(--cl-primary)` | Selected search result state, highlighted query text, focus rings |
| Destructive | `var(--cl-danger)` | Destructive actions only |

Accent reserved for: Active search result selection, highlighted match text, keyboard focus outlines (`var(--cl-focus)`).
