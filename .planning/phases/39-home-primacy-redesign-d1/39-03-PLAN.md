## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| PubSub `{:conversations_changed}` burst → HomeLive process mailbox | High-frequency internal broadcasts can flood the process if every tick triggers a re-query. |
| Count read → operator display | A failed count must not be misrepresented as a genuine "all caught up". |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
